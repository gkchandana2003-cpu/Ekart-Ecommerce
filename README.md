# 🛒 Ekart – E-commerce Web Application

## 📌 Overview
Ekart is a modern e-commerce web application built using React.js.  
It allows users to browse products, register/login, and manage their shopping cart smoothly.

---

## 🚀 Features
- 🔐 User Authentication (Firebase Login & Signup)
- 🛍️ Add to Cart functionality
- 🔄 Real-time cart updates
- 💾 Data stored using Local Storage
- 📱 Fully responsive design

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Firebase Authentication
- Context API
- Local Storage
